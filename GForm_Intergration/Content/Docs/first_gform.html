Creating Your First GForm

Home Form Collection Creating 1st GForm
  • Create a new UserWidget Blueprint with the parent class of 'GFormUserWidget'
  • Create your widget and form however you see fit, adding the respective GForm Widget with each question, here are all the Respective GForm Widgets and what the Google Form Question is: 
                        Short Answer - GForm Short Answer
                    Paragraph - GForm Paragraph
                    Multiple-Choice - GForm Multiple-Choice (Horizontal / Vertical Box)
                    Tick Box - GForm TickBox (Horizontal / Vertical Box)
                    Drop-Down - GForm Drop Down Combo Box String
                    Linear Scale - GForm Linear Scale (Horizontal / Vertical Box)
                    Rating - GForm Rating (Horizontal / Vertical Box)
                    Multiple-Choice Grid - GForm Multiple-Choice Grid
                    TickBox Grid - GForm TickBox Grid
                    Date - GForm Date
                    Time - GForm Time
                    Submit - GForm Submit Button
  • With your GForm Widget assembled, you will need to add the Multiple-Choice Boxes or TickBoxes for each of their respective Widgets, Multiple-Choice Boxes for the Multiple-Choice Horizontal Box for example. Add the same amount of boxes that exist in the Google Form.
  • With all that accomplished, the only thing to do now is to add the Unique information for each Widget, the place to add the information is found in 'GForm|Entry|EntryData'. Follow the instructions below for each Widget and complete the form!
    • Place the Unique Entry ID into the EntryID of the first element in the EntryData Array

Well done! You have just created your first GForm! Now add it to your viewport and test. If all goes well, when you click the submit button you should see the answers pop up in the Responses tab of Google Forms.
